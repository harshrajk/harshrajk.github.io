User Roles & Access Guide

Instructions for requesting access based on your role within the organization.

Overview

This document outlines the different roles in our organization, the types of access they may require, and the approval process for each. Please follow the relevant instructions for your role.

Roles & Access Details

1. Employee

General staff members performing day-to-day operational tasks.

Application Portal Access

Login to the main company portal for work-related tools.

Request Access
Approval: Direct Supervisor

File Server Access

Access shared documents and upload work files.

Submit Request
Approval: Supervisor + IT Security

2. Team Lead

Oversees a group of employees and manages team deliverables.

Reporting Dashboard Access

View team performance and operational analytics.

Fill Request Form
Approval: Department Head

File Server (Extended) Access

Manage team folders and control permissions.

Submit Extended Access Request
Approval: Department Head + IT Security

3. Manager

Manages multiple teams and coordinates cross-functional projects.

Admin Console Access

Approve user accounts, manage configurations.

Contact IT Admin
Approval: IT Director

External API Access

Integrate external tools and automate workflows.

Apply for API Key
Approval: CTO + Security Review

4. System Administrator

Responsible for system setup, maintenance, and security enforcement.

Full Admin Console Access

Manage all system settings and user roles.

Access Admin Console
Approval: None (Role-based)

Security Configuration Access

Modify firewalls, manage encryption keys, and set up monitoring tools.

Request Change Authorization
Approval: CTO

Approval Workflow

The approval process varies by role and type of access, but generally follows these steps:

  1. Submit your access request via the provided link or form.
  2. Direct supervisor or relevant authority reviews the request.
  3. Higher-level approvals may be required for sensitive access.
  4. IT team provisions the access after all approvals are complete.
